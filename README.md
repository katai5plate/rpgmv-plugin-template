# rpgmv-plugin-template

ツクール MV プラグインのテンプレートエディタ

## plan

- demo.json のような JSON からプラグインを生成する
- プラグインコマンドの実装も行われる
- `Document.currentScript` による存在確認の実装もする

## memo

- `Number[]` は出力時 `["1", "2", "3"]` の値にする
- `parent`, `text` は `parameters` 全 type に使用できる
